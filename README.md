## Overview

This project demonstrates how a Junior Security Operations Center (SOC) Analyst performs alert triage, validates security events, and determines when incidents should be escalated from Level 1 (L1) to Level 2 (L2).

The report focuses on practical SOC workflows used in real-world environments to improve incident response, documentation, and escalation accuracy.
